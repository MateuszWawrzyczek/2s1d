# Nawigacja i Dashboard

Sidebar prowadzi do wszystkich głównych obszarów aplikacji. Na dole menu widać
Twój e-mail, rolę i przycisk wylogowania.

## Dashboard

Dashboard pokazuje szybki przegląd systemu:

- liczbę przedmiotów,
- liczbę wypożyczonych przedmiotów,
- liczbę przeterminowanych wypożyczeń,
- liczbę kategorii,
- status bazy danych.

Status bazy odświeża się automatycznie. Jeśli widzisz `Błąd` albo `Nieznany`,
zgłoś problem administratorowi.

### Szybkie akcje

Kafle na Dashboardzie prowadzą do najczęstszych ścieżek:

- `Przeglądaj przedmioty`,
- `Obsłuż wypożyczenia`,
- `Skanuj kod QR`,
- `Dodaj przedmiot`,
- `Importuj z Excel`,
- `Drukuj etykiety`,
- `Raport przeterminowanych`.

## Co robi każdy tab

| Tab             | Po co służy                                                         |
| --------------- | ------------------------------------------------------------------- |
| `Przedmioty`    | Lista, filtry, szczegóły, edycja, lokalizacja, zdjęcia i delegacje. |
| `Kategorie`     | Drzewo kategorii oraz dodawanie, edycja i usuwanie gałęzi.          |
| `Statusy`       | Systemowe statusy i własne flagi przedmiotów.                       |
| `Delegacje`     | Przegląd wszystkich delegacji widocznych dla Twojego konta.         |
| `Wypożyczenia`  | Wnioski, zatwierdzanie, wydanie, zwroty i wypożyczenia zewnętrzne.  |
| `Skaner QR`     | Odczyt kodu QR albo kodu wpisanego ręcznie.                         |
| `Druk QR`       | Generowanie PDF z etykietami dla wybranych przedmiotów.             |
| `Import Excel`  | Import danych z arkusza.                                            |
| `Raporty`       | Raport przeterminowanych wypożyczeń.                                |
| `Użytkownicy`   | Konta, role i aktywność.                                            |
| `Grupy`         | Grupy opiekunów i członkostwo.                                      |
| `Powiadomienia` | Ustawienia przypomnień i historia zdarzeń.                          |
| `Logi audytu`   | Historia zmian w systemie.                                          |

## Zachowanie menu

- Na desktopie menu jest stale widoczne.
- Na mobile pojawia się przycisk menu i wysuwany panel.
- Jeśli nie masz uprawnień do danej pozycji, menu może ją ukryć.
- Jeśli pozycja jest widoczna, ale ekran kończy się `Brak uprawnień`, oznacza
  to blokadę po stronie roli.

## Co robić, gdy nie wiesz gdzie kliknąć

1. Zacznij od `Dashboard`.
2. Jeśli pracujesz na konkretnym sprzęcie, przejdź do `Przedmioty`.
3. Jeśli masz kod z etykiety, wybierz `Skaner QR`.
4. Jeśli chcesz zwrot albo wydanie, użyj `Wypożyczenia`.
